# Vanillatrade
Vanilla Trade Sambava

Site vitrine pour la promotion et la vente de Vanille Bourbon de Madagascar, provenant directement des producteurs locaux de Sambava.

 Aperçu

Vanilla Trade Sambava est un site web moderne qui permet de :

Présenter les différentes qualités de vanille

Mettre en valeur l'origine et la qualité du produit

Permettre aux clients de commander facilement via WhatsApp ou Mobile Money

Offrir une expérience utilisateur fluide et responsive

Technologies utilisées

HTML5

CSS3

JavaScript

Google Fonts

Font Awesome

Fonctionnalités

Preloader animé

Navigation responsive (menu hamburger)

Section héro avec call-to-action

Cartes produits avec prix

Section "Pourquoi nous choisir"

Copie rapide des numéros (WhatsApp, Mvola, Orange Money)

Design moderne et mobile-friendly
